projection(i) -- Map Projection Interface

SYNOPSIS

  • object box
  • object c2m zoom cx cy
  • object c2ref zoom cx cy
  • object c2ref zoom cx cy
  • object dim
  • object m2c zoom mx my....
  • object m2ref mx my
  • object ref2c zoom ref...
  • object ref2m ref...
  • object ref validate ref...

DESCRIPTION

A map projection(i) is an object that converts between mapcanvas(n) canvas coordinates and map coordinates given the map's projection. A projection(i) object will support the following kinds of coordinate system:

Canvas Units
The mapcanvas(n) refers to points on a map image using (cx,cy) pixel coordinates with the origin at the upper left; coordinates increase to the right and down.
Map Units
A location in map units is an (mx,my) coordinate pair. The units, and indeed the nature of the coordinate system, are entirely up to the projection(i) object. The mapref(n) projection, for example, uses a simple non-geographic scheme that's essentially just a rescaling of canvas units; when georeferencing is required, lat/lon is an obvious choice.
Map Reference Strings
A map reference string is a short string that identifies a point on the map; it's a human-readable equivalent to the (mx,my) map coordinates. The mapref(n) projection defines a simple alphanumeric string; when georeferencing is required, MGRS is an obvious choice.

Note that the mapcanvas(n) can display a given map at a variety of zoom factors. The zoom factor is an integer number between 1 and 300, interpeted as a percentage. At a zoom factor of 200, for example, there are twice as many pixels in each direction. The conversion between canvas coordinates and map coordinates must take the zoom factor into account.

The projection to be used depends on the projection of the map image, and on the desired map coordinate system. The mapref(n) projection has the advantage that it works with any map image, whether georeferenced or not.

Subcommands

Every projection(i) module must be an ensemble command with at least the following subcommands:

object box
Returns the bounding box of the map in map units.
object c2m zoom cx cy
Given a zoom factor and a canvas coordinate pair cx and cy, returns a map coordinate pair as a list of two coordinates mx and my.
object c2ref zoom cx cy
Given a zoom factor and a canvas coordinate pair cx and cy, returns a map reference string.
object c2ref zoom cx cy
Given a zoom factor and a canvas coordinate pair cx and cy, returns a map location for purposes of display. How a location is displayed to a user is projection dependant.
object dim
Returns the dimensions of the map in map units.
object m2c zoom mx my....
Given a zoom factor and one or more map coordinate pairs mx and my, returns a list of canvas coordinate pairs as a flat list of coordinates cx cy....
object m2ref mx my
Given a map coordinate pair mx and my, returns a map reference string.
object ref2c zoom ref...
Given a zoom factor and one or more map reference strings, returns a list of canvas coordinate pair as a flat list of coordinates cx cy....
object ref2m ref...
Given one or more map reference strings as arguments, returns a corresponding list of mx and my map coordinates.
object ref validate ref...
Given one or more map reference strings as arguments, validates each. Throws an error with -errorcode INVALID if any reference string is invalid.
