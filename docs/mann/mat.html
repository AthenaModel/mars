DESCRIPTION

mat(n) defines a number of operations on Tcl matrices. A Tcl matrix is a list of lists in row-major order, indexed i = 0,..., m-1 and j = 0,..., n-1. Thus, the following code defines an initialized matrix of 2 rows and 3 columns: 

set matrix {
    {1 2 3}
    {4 5 6}
}

Matrices can be entered explicitly, as shown; alternatively, the mat new command can create a matrix of arbitrary size. In general, the elements of a matrix can be of any type, although some of the operations defined by this module assume them to be numeric.

The elements of a matrix are accessed using the standard Tcl commands lindex and lset: 

# Print element i,j
puts [lindex $matrix $i $j]

# Set element i,j to 7
lset matrix $i $j 7

Note that Tcl lists are represented internally as extensible arrays; consequently, access to matrix elements is reasonably quick.

COMMANDS

mat new m n ?initval?

Returns a new matrix of m rows and n columns. If initval is given, all elements will be set to it; otherwise all elements will contain the empty string. For example, the following code creates a 2*3 matrix of zeroes: 

set matrix [mat new 2 3 0]
mat rows matrix
Returns the number of rows in the matrix.
mat cols matrix
Returns the number of columns in the matrix.
mat rowvec matrix i
Returns the ith row of matrix as a vec(n).
mat colvec matrix j
Returns the jth column of matrix as a vec(n).
mat equal mat1 mat2
Returns 1 if the two matrices have the same dimensions and corresponding elements are numerically equal, and 0 otherwise.
mat add mat1 mat2
Returns the sum of the two matrices mat1 and mat2.
mat sub mat1 mat2
Returns the difference of the two matrices mat1 and mat2.
mat scalarmul matrix constant
Returns the product of the matrix with a scalar constant.
mat format matrix fmtstring
Returns a copy of the matrix which each element formatted using the standard Tcl format command and the given format string.
mat pprint matrix ?rlabels? ?clabels?

Returns a pretty-printed text string of the elements in the matrix. If specified, rlabels is a list of mat rows row labels, and clabels is a list of mat cols column labels; otherwise, the row and column labels are defaulted to the row and column numbers. The matrix elements are included "as is", with no additional formatting.

For example, the command 

mat pprint {
   {1 2 3}
   {4 5 6}
}

returns the following string: 

       Col 0  Col 1  Col 2
Row 0      1      2      3
Row 1      4      5      6
mat pprintf matrix format ?rlabels? ?clabels?

This is a variant of mat pprint; before the matrix is pretty-printed, the elements are formatted using the standard Tcl command format and the specified format string. For example, the following code pretty-prints a numeric matrix containing decimal numbers between 0.000 and 100.000: 

puts [mat pprintf $matrix "%7.3f"]

Note that empty elements are passed through to the output.

mat pprintq matrix quality ?rlabels? ?clabels?

This is a variant of mat pprint; the matrix is assumed to contain numeric values of a quality(n) type. The quality object is used to format each element as follows: 

<shortname>=<value>

The value is formatted using the quality object's format subcommand; the shortname is the short symbolic name corresponding to the value.

mat numerize matrix quality
Given a matrix whose elements are symbols of the specified quality object, returns a matrix in which all symbols have been converted to their numeric equivalents. Any pre-existing numeric elements in the matrix are retained.
mat filter matrix command
Given a matrix, pass each element to command and return the matrix of results. The command is an arbitrary Tcl command expecting one additional argument.

SEE ALSO

vec(n), mat3d(n).

ENVIRONMENT

mat(n) requires Tcl 8.5 or later.

To use mat(n) in a Tcl script, the environment variable TCLLIBPATH must include the parent of the marsutil(n) library directory.
