isearch(n) -- Incremental Search, submodule of marsgui(n)

SYNOPSIS

package require marsgui 3.0.23
namespace import ::marsgui::*
  • isearch enable w
  • isearch enable w
  • isearch logger w ?cmd?

DESCRIPTION

isearch(n) is a module that enables incremental searching in Tk text widgets, and in components derived from them. Incremental searching is a mode where the text widget shows the user the found text as the user types. See BINDINGS to see how it is used.

COMMANDS

This module defines the following commands:

isearch enable w

Enables incremental searching for the Tk text widget named w. If w is all, then incremental searching is enabled on the Text binding, and thus effects all text widgets in the application (this might not be a good idea).

Typically, isearch enable is called for the widget just after it is created.

isearch enable w

Disables incremental searching for the Tk text widget named w. If w is all, then incremental searching is disabled on the Text binding.

There's generally no reason to disable incremental searching once it is enabled.

isearch logger w ?cmd?

Sets/queries a logger command for the Tk text widget named w, or for all isearch-enabled text widgets if w is all. The cmd is a Tcl command prefix that takes one additional argument, the message.

The logger command will usually write the message to a status line below the text widget; it keeps the user apprised of the progress of the search. This, however, is optional; incremental searching will work perfectly well even if no logger is defined.

BINDINGS

Incremental searching is activated by the following keystrokes:

  • Ctrl+F -- Search forwards from the insertion point
  • Ctrl+R -- Search backwards from the insertion point

Once in incremental search mode, the widget's normal bindings are suspended, and the following bindings are enabled.

First, the user may begin typing the search target, and the widget will search for it as the user types.

Next, the Backspace key will back up in the search, deleting characters from the search target.

Next, pressing Ctrl+F will find the next occurrence of the currently found search target, and pressing Ctrl+R will find the previous occurrence.

Note that if Ctrl+R or Ctrl+F is pressed immediately after entering incremental search mode, the widget will search for the search target from the previous search. In other words, to search for the same thing again, press Ctrl+F twice in a row.

Next, incremental search mode will terminate if:

  • The user clicks with the mouse in the text widget.
  • The text widget loses the keyboard focus.
  • The user presses Escape, Return, Tab, or any of cursor movement keys. Other keys, including control, meta, and alt-key combinations, will have no effect.

    ENVIRONMENT

    marsgui(n) requires Tcl 8.5 or later. To use marsgui(n) in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
