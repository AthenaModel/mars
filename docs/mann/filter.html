SYNOPSIS

namespace import ::marsgui::*

  •     -filtercmd cmd
  •     -filtertype value
  •     -ignorecase flag
  •     -msgcmd cmd
  • $filter check string

DESCRIPTION

This widget provides a text filter control for doing exact, wildcard and regular expression-based filtering, both inclusive and exclusive, of arbitrary text. The widget contains a sieve icon which pops up a menu, an entry field (with command history) for typing filter text, and a clear button.

Filtering is triggered when <Return> is pressed in the entry field, when the field is cleared, and when the search type and inclusive/exclusive match flag is set. When filtering is triggered, the client's -filtercmd is called. The -filtercmd will typically pass one or more snippets of text to the check method, and display them or not depending on the result.

The Sieve Icon Menu

The first set of entries on the menu allows the user to select the type of the filter text:

  • Exact -- An arbitrary text string
  • Incremental -- An arbitrary text string, matched incrementally as the user enters each character.
  • Wildcard -- A string containing wildcard characters, e.g., "*"
  • Regexp -- A regular expression

The second set is a single flag, Ignore Case.

The third set indicates whether filtration is inclusive or exclusive:

  • Include Matches -- data items which match the filter text pass the filter.
  • Exclude Matches -- data items which do not match the filter text pass the filter.

COMMANDS

This module defines the following command:

filter pathName ?options...?

Creates a new filter widget named pathName.

The filter widget may be created with all the standard Tk frame widget options, plus the following filter(n)-specific options:

-width chars
Specifies the character width of the filter(n)'s entry field. Defaults to 20.
-filtercmd cmd

Specifies a command to be called whenever filtration is triggered; the command can do anything at all, but will usually use check to filter one or more snippets of text.

The -filtercmd will be called when <Return> is pressed in the widget's text field, when any character is pressed (in incremental mode), when the text field is cleared, when an option is changed on The Sieve Icon Menu, and when the -filtertype or -ignorecase options are changed by the application.

-filtertype value
Explicitly sets the filter type to one of exact (the default), incremental, wildcard, or regexp, and calls the -filtercmd.
-ignorecase flag
Explicitly sets the ignore case flag (the default is false), and calls the -filtercmd.
-msgcmd cmd
A command for reporting messages to the user; if defined, cmd will be called with a message appended to it as a single argument whenever filter(n) wishes to report a message.

INSTANCE COMMAND

$filter check string
Checks the string against the current filter text and settings, and returns 1 if the string passes the filter and 0 otherwise.

ENVIRONMENT

marsgui(n) requires Tcl 8.5 or later.

To use marsgui(n) in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
