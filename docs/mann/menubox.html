DESCRIPTION

menubox(n) is a ttk::combobox(n) widget tailored to be used as a read-only pull-down widget.

COMMANDS

This module defines the following command:

menubox pathName ?options...?

Creates a new menubox(n) widget named pathName.

The menubox(n) widget may be created with any of the ttk::combobox options, as well as the following:

-command cmd
Specifies a command to be called, with no additional arguments, whenever the user selects an item from the menubox's pulldown list.

INSTANCE COMMAND

The instance command has the same subcommands as the ttk::notebook widget, starting with the following:

win configure option value...
Sets the values of the specified options.
win cget option
Returns the value of the specified option.

ENVIRONMENT

marsgui(n) requires Tcl 8.5 or later.

To use marsgui(n) in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
