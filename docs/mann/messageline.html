SYNOPSIS

package require marsgui 3.0.23
namespace import ::marsgui::*
  • messageline pathName ?options...?
  •     -font fontspec
  •     -blankinterval msecs
  • $messageline puts msg

DESCRIPTION

The messagelin(n) widget provides a message line to be used at the bottom of application windows. The messageline will usually be blank. On request it will display a single line of text, which will remain until replaced or until a programmer-defined interval elapses.

COMMANDS

This module defines the following command:

messageline pathName ?options...?

Creates a new messageline(n) widget named pathName.

The messageline widget may be created with the following options:

-font fontspec
Specifies the widget's text font.
-blankinterval msecs
If set to 0, messages written to the message line will be displayed until they are replaced by some other non-blank message. Otherwise, the messageline will be blanked whenever a message has been displayed for longer than msecs milliseconds.

INSTANCE COMMAND

$messageline puts msg
Displays msg, which should be a single line of text, on the message line.

ENVIRONMENT

marsgui(n) requires Tcl 8.5 or later.

To use marsgui(n) in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
