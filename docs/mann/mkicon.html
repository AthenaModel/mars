NAME

mkicon(n) -- Tk Icon Maker, submodule of marsgui(n)

SYNOPSIS

package require marsgui 3.0.23
namespace import ::marsgui::*
  • mkicon cmd charmap colors ?suffix mod...?
  • mkiconfile name fmt charmap colors ?suffix mod?

DESCRIPTION

mkicon(n) provides a commands for creating families of icon images from text input. The images can be created as Tk image commands, or as image files.

COMMANDS

This module defines the following commands:

mkicon cmd charmap colors ?suffix mod...?

This command creates a Tk image given the charmap, a text representation of the pixels, and colors, which relates characters in the charmap to specific colors. The cmd is the name to be given to the new image command; if it is the empty string, a name is automatically generated. If it is not a fully-qualified name, it is presumed to be in the caller's namespace.

The charmap is a list of strings, each string denoting one row of pixels. The strings are usually formatted in successive lines. Each character represents a color. The colors argument is simply a dictionary of characters and Tk color specifications, with one addition: the pseudo-color name trans. Pixels mapped to trans will be transparent.

The caller can specify a sequence of suffix and mod arguments to create a family of icons based on the same charmap, where mod is a dictionary of characters and Tk color specifications that modifies the definitions in colors. The first icon will be created as described above, and the subsequent ones will be created by appending suffix to the cmd and merging mod with colors.

The command returns a list of the names of the newly created image commands.

For example, the follow command defines an image similar to a mouse pointer arrow with a black foreground and a transparent background, along with a gray icon to be used when the icon is disabled. 

mkicon arrow {
    X..........
    XX.........
    XXX........
    XXXX.......
    XXXXX......
    XXXXXX.....
    XXXXXXX....
    XXXXXXXX...
    XXXXXXXXX..
    XXXXXXXXXX.
    XXXXXXXXXXX
    XXXXXXX....
    XXX.XXXX...
    XX..XXXX...
    X....XXXX..
    .....XXXX..
    ......XXXX.
    ......XXXX.
} { . trans X black } d { X gray }
mkiconfile name fmt charmap colors ?suffix mod?
This command takes the same input as mkicon, but saves the new images to a series of disk files using name and suffix instead of creating a Tk image. The files will have the specified format, typically gif or png. Returns a list of the absolute file names of the new files.

ENVIRONMENT

This package requires Tcl/Tk 8.5 or later.

To use this package in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
