DESCRIPTION

The stripchart(n) widget plots from one to ten series of data as line plots against a single X axis (usually time stamps). Individual series of data can be updated individually.

OPTIONS

The widget has the following options.

-closeenough pixels
Specifies a floating-point value indicating how close the mouse cursor must be to an item before it is considered to be "inside" the item. Defaults to 1.0.
-height pixels
Specifies the height of the chart window.
-smooth flag
If flag is true, the line plotted for each data series will be smoothed. This yields a more attractive appearance, but is slower. The default is false.
-title text
Specifies an overall title for the chart. If text is the empty string (the default), no title is drawn.
-titlepos anchor
Specifies where to put the title; anchor may be n or s to center the title at the top or bottom. Defaults to n.
-width pixels
Specifies the widget of the chart window.
-xformatcmd cmd

Specifies a command for formatting X-axis values; the command should take one additional argument, the value to format. Defaults to: 

    format %g
-xmax value
Specifies the maximum bound for the X-axis, if it's bigger than the maximum X value in the plotted data. Specify "" to use the maximum data value in any event.
-xmin value
Specifies the minimum bound for the X-axis, if it's smaller than the minimum X value in the plotted data. Specify "" to use the minimum data value in any event.
-xtext text
Specifies a label for the X-axis. If text is the empty string (the default), no label is drawn.
-ytext text
Specifies a label for the Y-axis. If text is the empty string (the default), no label is drawn.
-yformatcmd cmd

Specifies a command for formatting y-axis values; the command should take one additional argument, the value to format. Defaults to: 

    format %g

COMMANDS

This module defines the following command:

stripchart pathName ?options...?
Creates a new stripchart widget named pathName.

INSTANCE COMMAND

win cget option
Returns the value of the specified option.
win clear
Deletes all plotted data series.
win configure option value...
Sets the values of the specified options. The widget will redraw the chart accordingly.
win plot name ?options...?

Plots a series of data values. The name is used to identify the series for future updates.

The options are as follows:

-data coords
Specifies a flat list of X/Y pairs.
-label text
Specifies a label for this series of data. The label is used in the legend. If no -label is given, the series name is used instead.
-rmin min
-rmax max
These options specify the Y-axis range bounds for the series--that is, not the min and max values in the data, but the logical min and max value for the data type. Strict percentages, for example, have a range from 0.0 to 100.0. If one of these options is omitted or explicitly set to the empty string, the relevant bound is computed from the data.

VIRTUAL EVENTS

The stripchart(n) widget generates the following virtual events.

<<Context>>

This event is generated when the user right-clicks or control-clicks on the chart. It supports the following substitutions: %x, %y, %X, %Y, and %d.

The value of the %d substitution is a list whose first element indicates what was clicked on; the remaining items in the list vary depending on the first element. The current possibilities are as follows:

none
The user right-clicked on nothing in particular.
plot x y
The user right-clicked in the plot area, though not on any particular data series; x and y are the data coordinates corresponding to the location of the mouse-click.
line x y series
The user right-clicked on the plot line for a particular data series; x and y are the data coordinates corresponding to the location of the mouse-click.

ENVIRONMENT

This package requires Tcl/Tk 8.5 or later.

To use this package in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
