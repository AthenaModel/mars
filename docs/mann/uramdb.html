DESCRIPTION

uramdb(n) defines a parser for uramdb(5) files. uramdb(n) loads uramdb(5) data into an sqldocument(n) or other SQLite3 database. uramdb(n) adheres to the sqlsection(i) interface.

CREATE TABLE uramdb_c (
    -- Concern names
    c    TEXT PRIMARY KEY          -- Symbolic name
);

CREATE TABLE uramdb_a (
    -- Actor names
    a    TEXT PRIMARY KEY          -- Symbolic name
);

CREATE TABLE uramdb_n (
    -- Neighborhood names
    n    TEXT PRIMARY KEY          -- Symbolic name
);

CREATE TABLE uramdb_g (
    -- FRC group names and data
    g     TEXT PRIMARY KEY,        -- Symbolic group name
    gtype TEXT                     -- CIV, FRC, ORG   
);

CREATE TABLE uramdb_civ_g (
    -- CIV group names and data
    g    TEXT PRIMARY KEY,         -- Symbolic group name
    n    TEXT,                     -- Nbhood of reference
    pop  INTEGER DEFAULT 0 
);

CREATE TABLE uramdb_frc_g (
    -- FRC group names and data
    g    TEXT PRIMARY KEY          -- Symbolic group name
);

CREATE TABLE uramdb_org_g (
    -- ORG group names and data
    g    TEXT PRIMARY KEY          -- Symbolic group name
);


CREATE TABLE uramdb_mn (
    -- Pairwise neighborhood data
    m          TEXT,                     -- Symbolic nbhood name
    n          TEXT,                     -- Symbolic nbhood name

    proximity  TEXT DEFAULT 'REMOTE',    -- eproximity
    
    PRIMARY KEY (m, n)
);

CREATE TABLE uramdb_hrel (
    -- HREL: group-to-group horizontal relationships
    f      TEXT,                     -- Symbolic group name
    g      TEXT,                     -- Symbolic group name

    hrel   DOUBLE DEFAULT 0.0,       -- Horizontal relationship
    
    PRIMARY KEY (f, g)
);

CREATE TABLE uramdb_vrel (
    -- VREL: group-to-actor vertical relationships
    g      TEXT,                     -- Symbolic group name
    a      TEXT,                     -- Symbolic actor name

    vrel   DOUBLE DEFAULT 0.0,       -- Vertical relationship
    
    PRIMARY KEY (g, a)
);


CREATE TABLE uramdb_sat (
    -- SAT: Satisfaction levels
    g          TEXT,                     -- Symbolic group name
    c          TEXT,                     -- Symbolic concern name

    sat        DOUBLE DEFAULT 0.0,       -- Initial satisfaction
    saliency   DOUBLE DEFAULT 1.0,       -- Saliency

    PRIMARY KEY (g, c)
);

CREATE TABLE uramdb_coop (
    -- COOP: CIV group cooperation with FRC group
    f        TEXT,                     -- Symbolic group name
    g        TEXT,                     -- Symbolic group name

    coop     DOUBLE DEFAULT 50.0,      -- Cooperation
    
    PRIMARY KEY (f, g)
);

COMMANDS

This module defines the following command:

uramdb loadfile dbfile ?db?

Parses the content of the file named dbfile, and stores the data in the relevant tables in the sqldocument(n) called db.

If db is specified, the schema must already be initialized. The relevant tables will be cleared prior to loading the data.

If no db is specified, loadfile will create, initialize, and return an in-memory sqldocument(n) object. In this case, it is the caller's responsibility to destroy the database object when it is no longer needed.

uramdb load text ?db?
Identical to loadfile, but parses uramdb(5) text from string text rather than a file.
uramdb mkperfdb db ?options...?

Populates the uramdb(5) tables in the specified db with automatically-generated data for the purpose of measuring worst-case performance. The options determine the size of the database:

-actors num
The number of actors in the playbox.
-nbhoods num
The number of neighborhoods in the playbox.
-civgroups num
The number of civilian groups per neighborhood.
-frcgroups num
The number of force groups in the playbox.
-orggroups num
The number of organization groups in the playbox.

The generated data will have the following characteristics:

  • All neighborhood proximities are HERE or NEAR.
  • All satisfaction levels are 0.0.
  • All saliencies are 1.0.
  • All cooperation levels are 50.0.
  • All populations are 1000.
  • All group relationships are 1.0, 0.5, or -0.5. Each group will have a 1.0 relationship with itself and 0.5 or -0.5 with all other groups.
  • All vertical relationships are 0.5 or -0.5.
uramdb loader db uram

Loads the uramdb(5) data found in the sqldocument(n) object db into uram, an instance of uram(n). This command is to be used as a uram2(n) -loadcmd, like so: 

uram ::myuram \
    -loadcmd [list ::simlib::uramdb loader $db] \
      .
      .
      .

where $db is the name of the sqldocument(n) containing the data. Note that calling it directly simply won't work, as uram(n) won't be expecting it.

uramdb sqlsection subcommand
uramdb(n) is an sqlsection(i), as it defines schema entities (see DATABASE SCHEMA). See sqlsection(i) for a description of this method's subcommands.

ENVIRONMENT

This package requires Tcl 8.5 or later.

To use this package in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
