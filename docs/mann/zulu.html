zulu(n) -- Zulu-time Conversion, submodule of marsutil(n)

SYNOPSIS

package require marsutil 3.0.23
namespace import ::marsutil::*
  • zulu fromsec seconds
  • zulu tosec zulutime
  • zulu validate zulutime

DESCRIPTION

zulu(n) defines a number of operations on "zulu-time" strings. A zulu-time string is a UTC date-and-time string with this format: 

ddhhmmZmmmyy

where

  • "dd" is the day of the month, 01 to 31
  • "hh" is the hour of the day, 00 to 23
  • "mm" is the minute of the hour, 00 to 59
  • "Z" is a literal Z
  • "mmm" is the abbreviated month name, JAN, FEB, MAR, etc.
  • "yy" is the two-digit year.

This module defines conversions from zulu-time strings to Unix clock seconds and vice-versa.

Note: CBS understands a two-digit year "yy" to fall in the range 1950 to 2049. This module is based on Tcl's clock command, which uses the range 1969 to 2068 instead.

COMMANDS

zulu fromsec seconds
Given a standard Unix time in clock seconds, returns the zulu-time string. Seconds within the minute are truncated.
zulu tosec zulutime
Given a zulutime string, returns the standard Unix time in clock seconds.
zulu validate zulutime
Validates the zulutime, throwing an error with error code INVALID for invalid strings and returning the canonical form (all uppercase) otherwise.

ENVIRONMENT

zulu(n) requires Tcl 8.5 or later.

To use zulu(n) in a Tcl script, the environment variable TCLLIBPATH must include the parent of the marsutil(n) library directory.
