DESCRIPTION

The oohelpers(n) package contains commands that make using TclOO a bit easier.

GENERAL COMMANDS

This module defines the following commands to be used with TclOO classes and objects.

isa cls obj
Returns 1 if obj is an instance of cls or one of its subclasses, and 0 otherwise.
isancestor parent child
Returns 1 if parent is an ancestor class of the child class (including being a direct superclass), and 0 otherwise.

CLASS-DEFINITION COMMANDS

This module defines the following commands to be used in class definitions or with oo::define:

meta name value

Defines a class metadata variable, a constant value that can be retrieved from the class object and from its instances by calling the method of the same name. For example, 

oo::class create myclass {
    meta title "The Class's Title"
}

is equivalent to 

oo::class create myclass {
    self method title {} {
        return "The Class's Title"
    }
    method title {} {
        return [[self class] title]
    }
}
metadict name dict

Defines a class metadata variable whose value is a dictionary. Such variables are just like normal meta variables, except that the automatically generated methods have an optional key argument, allowing for dictionary look ups. For example, 

oo::class create myclass {
    meta lookup { 
        A 4.0
        B 3.0
        C 2.0
        D 1.0
        F 0.0
    }
}

set dict [myclass lookup]
set points [myclass lookup B]

set obj [myclass new]
set dict [$obj lookup]
set points [$obj lookup B]
typemethod method arglist body

Deprecated. In class definitions, the syntax "self method" is equivalent to "typemethod" as defined here.

Defines a method on the class's class object. In other words, 

oo::class create myclass {
    typemethod init {} { ... }
}

is equivalent to 

oo::class create myclass
oo::objdefine myclass {
    typemethod init {} { ... }
}

BODY COMMANDS

This module defines the following commands for use in method bodies (including constructor and destructor bodies):

mymethod method ?args...?
Returns a command prefix consisting of a call to the current object's named method plus any additional arguments. The command prefix can be called by other objects even if the method is unexported.

ENVIRONMENT

This package requires Tcl 8.6 or later.
