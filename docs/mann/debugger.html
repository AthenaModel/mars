SYNOPSIS

package require marsgui 3.0.23
namespace import ::marsgui::*
  • debugger pathName ?options...?
  • debugger new ?options...?
  • debugger debug ?command...?
  • $debugger configure option value...
  • $debugger cget option
  • $debugger edit command

DESCRIPTION

marsgui(n) contains general-purpose (i.e., non-application-specific) GUI code for use in Mars. This module defines the ::marsgui::debugger widget, a toplevel window which can be invoked to help debug problems in a a Tcl/Tk application.

At present, the ::marsgui::debugger provides the following features:

  • A cli(n) which provides command-line access to the main Tcl interpreter.
  • A winbrowser(n), which provides information on any window in the Tk window tree.
  • A cmdbrowser(n), which provides information on any command known to the Tcl interpreter.
  • A modeditor(n), which allows the user to edit code in the running application, source it back in, and save it to disk as a mod file.

OPTIONS

The debugger widget has the same options as the standard Tk "toplevel" widget; in addition, it provides the following:

-app flag
Read-only. The flag must be a Boolean value. If it is true, the created window assumes that it is the main window in the application; closing the window will terminate the program. The option defaults to false.>
-editordefaultdir path
Specifies the default directory for the editor tab's "File Save" dialog. If unset, the dialog will open in the current working directory.
-editorformatcmd command
Specifies the name of a command used to format an edited mod before saving it. The command takes two additional arguments, the file name and the code to format. It should return the formatted text. The application can use this to add boilerplate to the saved file.

COMMANDS

This module defines the following command:

debugger pathName ?options...?
Creates a new toplevel debugger window named pathName.
debugger new ?options...?
Creates a new toplevel debugger window, with the specified options.
debugger debug ?command...?
Pops up a debugger, creating one if necessary, and attempts to load command... into the debugger's mod editor.

INSTANCE COMMAND

$debugger configure option value...
Sets the values of the specified options.
$debugger cget option
Returns the value of the specified option.
$debugger edit command
Loads the command's definition into the debugger's mod editor.

ENVIRONMENT

This package requires Tcl/Tk 8.5 or later.

To use this package in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
