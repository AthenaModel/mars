SYNOPSIS

package require marsgui 3.0.23
namespace import ::marsgui::*
  • datagrid pathName ?options...?
  • $datagrid configure option value...
  • $datagrid cget option
  • $datagrid label r c ?options...?
  • $datagrid value r c ?options...?

DESCRIPTION

marsgui(n) contains general-purpose (i.e., non-application-specific) GUI code for use in Mars.

This module defines the ::marsgui::datagrid widget, which makes it easier to create an array of labeled values.

datagrid(n) is a drop-in replacement for the standard Tk frame, with a few additional subcommands for adding labels and values in specific grid cells. Cells are indexed by row and column, as for Tk's "grid" geometry manager. Items placed in the datagrid are displayed as follows:

  • Labels are displayed using the standard label font.
  • Values are displayed using the marsgui(n) "codefont" (a monospace font).
  • Labels and values are aligned left.
  • Any extra space is allocated to the last row and column.

Note that a cell may contain only one item.

COMMANDS

This module defines the following command:

datagrid pathName ?options...?
Created a new datagrid(n) widget; the options are as for Tk's frame(n) widget.

INSTANCE COMMAND

This module defines the following command:

$datagrid configure option value...
Sets the values of the specified options.
$datagrid cget option
Returns the value of the specified option.
$datagrid label r c ?options...?

Places a Tk label widget in the cell at row r, column c. The options determine what is displayed, e.g., 

$datagrid label 0 0 -text "My Label:"
$datagrid value r c ?options...?

Places a Tk label widget in the cell at row r, column c, and configures it to display a data value. The options determine what is displayed, e.g., 

$datagrid value 0 1 -width 12 -textvariable ::valueBuffer

ENVIRONMENT

marsgui(n) requires Tcl 8.5 or later.

To use marsgui(n) in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
