coverage(n) -- Coverage Function, submodule of simlib(n)

SYNOPSIS

package require simlib 3.0.23
namespace import ::simlib::*
  • coverage validate func
  • coverage eval func personnel population

DESCRIPTION

coverage(n) validates and evaluates personnel coverage functions, as defined in the Mars Analyst's Guide. A coverage function computes the fraction of a civilian population affected, or "covered", by some number of personnel engaged in some activity. A coverage function is specified as a pair of parameters c and d, e.g., {25 1000}. This function says that to achieve two-thirds coverage of a population requires a troop density of at least 25 troops per 1000 people in the population. In other words, if the population is 100,000 then (given this function) a force of 2,500 personnel will affect two-thirds them.

COMMANDS

coverage validate func

Validates the function specification, func. It must be a Tcl list of two numbers, c and d; c is some number of troops, and must be non-negative; d is some quantity of people in the population, e.g., 1000, and must be strictly positive.

This command returns the func if it is valid and throws an error with error code INVALID if it is invalid.

coverage eval func personnel population
Returns the coverage fraction given the function specifier, func, the number of personnel present, and the size of the population. The fraction will be 0.0 for when there are 0 personnel, and asymptotically approaches 1.0 as the number of personnel increases.

ENVIRONMENT

This package requires Tcl 8.5 or later.

To use this package in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
