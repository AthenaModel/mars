DESCRIPTION

vec(n) defines a number of operations on Tcl vector. A Tcl vector is just a Tcl list, indexed i = 0,..., n-1; this module doesn't distinguish between row and column vectors, which should more properly be represented using mat(n).

For example, the following code creates a 3-tuple: 

set vector {1 2 3}

Vectors can be entered explicitly, as shown; alternatively, the vec new command can create a vector of arbitrary size. In general, the elements of a vector can be of any type.

The elements of a vector are accessed using the standard Tcl commands lindex and lset: 

# Print element i
puts [lindex $vector $i]

# Set element ito 7
lset vector $i 7

Note that Tcl lists are represented internally as extensible arrays; consequently, access to vector elements is reasonably quick.

COMMANDS

vec new n ?initval?

Returns a new vector with n elements. If initval is given, all elements will be set to it; otherwise all elements will contain the empty string. For example, the following code creates a vector of six zeroes: 

set vector [vec new 6 0]
vec size vector
Returns the number of elements in the vector.
vec equal vec1 vec2
Returns 1 if the two vectors have the same size and corresponding elements are numerically equal, and 0 otherwise.
vec add vec1 vec2
Returns the sum of the two vectors.
vec sub vec1 vec2
Returns the difference of the two vectors.
vec scalarmul vec constant
Returns the product of the vector and the scalar constant.
vec normalize vec

Given a vector of non-negative numbers not all zero, returns a vector of numbers between 0 and 1 which sum to 1.0. The values are normalized by dividing by sum of the numbers in the vector.

An error is thrown if the sum of the values is less than or equal to zero.

vec format vector fmtstring
Returns a copy of the vector which each element formatted using the standard Tcl format command and the given format string.
vec numerize vector quality
Given a vector whose elements are symbols of the specified quality object, returns a vector in which all symbols have been converted to their numeric equivalents. Any pre-existing numeric elements in the vector are retained.
vec pprint vector labels

Returns a pretty-printed text string of the elements in the vector. labels is a list of vec size labels. The vector elements are included "as is", with no additional formatting.

For example, the command 

vec pprint {1 2 3} {A B C}

returns the following string: 

A 1
B 2
C 3
vec pprintf vector labels format

This is a variant of vec pprint; before the vector is pretty-printed, the elements are formatted using the standard Tcl command format and the specified format string. For example, the following code pretty-prints a numeric vector containing decimal numbers between 0.000 and 100.000: 

puts [vec pprintf $vector $labels "%7.3f"]
vec pprintq vector labels quality

This is a variant of vec pprint; the vector is assumed to contain numeric values of a quality(n) type. The quality object is used to format each element as follows: 

<shortname>=<value>

The value is formatted using the quality object's format subcommand; the shortname is the short symbolic name corresponding to the value.

SEE ALSO

mat(n).

ENVIRONMENT

vec(n) requires Tcl 8.5 or later.

To use vec(n) in a Tcl script, the environment variable TCLLIBPATH must include the parent of the marsutil(n) library directory.
