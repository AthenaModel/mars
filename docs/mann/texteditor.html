texteditor(n) -- Text Editor Widget, submodule of marsgui(n)

SYNOPSIS

package require marsgui 3.0.23
namespace import ::marsgui::*

DESCRIPTION

marsgui(n) contains general-purpose (i.e., non-application-specific) GUI code for use in Mars. This module defines the ::marsgui::texteditor, a tailored text(n) widget. texteditor(n) is a drop in replacement for the Tk text widget, with these behaviors added:

  • The widget responds to the <<Undo>> and <<SelectAll>> virtual events. marsgui(n) binds these to Ctrl+Z and Ctrl+Shift+A, respectively.
  • The Tab key inserts spaces rather than tab characters, and tab stops are every four spaces.

The options and subcommands are simply those of the text(n) widget, and so won't be documented here.

ENVIRONMENT

This package requires Tcl/Tk 8.5 or later.

To use this package in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
