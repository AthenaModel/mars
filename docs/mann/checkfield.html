SYNOPSIS

package require marsgui 3.0.23
namespace import ::marsgui::*
  • checkfield pathName ?options...?
  •     -changecmd command
  •     -state state
  • pathName cget option
  • pathName configure option value...
  • pathName get
  • pathName set value ?-silent?

DESCRIPTION

The checkfield(n) widget is a ttk_checkbutton(n) configured as a field(i) data entry field. It is used for entering boolean flags.

COMMANDS

This module defines the following commands.

checkfield pathName ?options...?

Creates a new checkfield widget named pathName, and returns pathName. The widget supports the standard Ttk widget options, as described on the ttk_checkbutton(n) man page, but not the -variable, -onvalue, -offvalue, and -command, as these are used (and hidden) by the wrapper. In particular, the -text, -image, and -compound options can be used to configure the widget's displayed text or image.

-changecmd command
From field(i). Specifies a command to be called whenever the field's value changes, for any reason whatsoever (including explicit calls to set). The new value is appended to the command as an argument.
-state state
From field(i). Sets the state to normal or disabled. If disabled, the field cannot be edited by the user.

INSTANCE COMMAND

checkfield(n) widgets have the following subcommands:

pathName cget option
From field(i). Returns the value of the specified option.
pathName configure option value...
From field(i). Sets the value of one or more of the object's options.
pathName get
From field(i). Returns the field's current value.
pathName set value ?-silent?
From field(i). Explicitly sets the field's value to any valid boolean value; the value will be saved as 1 or 0 internally. If the value has really changed, the -changecmd is called, unless the -silent option is given. Note that successive calls to set with the same value will result in at most one call to the -changecmd.

ENVIRONMENT

marsgui(n) requires Tcl 8.5 or later.

To use marsgui(n) in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
