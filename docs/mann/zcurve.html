DESCRIPTION

A zcurve is a piecewise-linear function with three segments, called a Z-curve because of its shape. Here's a stereotypical example, Z(x): 

        |            a                    |
     lo |------------+                    |     
        |             \                   |
        |              \                  |
        |               \                 |
        |                \                |
        |                 \               |
        |                  \              |
        |                   \             |
        |                    +------------| hi
        |                    b            |
        +----------------+----------------+
       -x                0               +x

The function is defined for all values of x; the only real constraint is that a <= b. Note that the name "lo" signifies the Y-value at the low end of the X-axis, and "hi" signifies the Y-value at the high end of the X-axis. If lo < hi the curve that slopes up from left to right instead of down.

If a equals b (within an epsilon) then Z(a) equals Z(b) equals the average of lo and hi.

Practically speaking, different kinds of Z-curves will have standard limits for the four parameters; to validate these, define an instance of zcurve and specified the desired -xmin, -xmax, -ymin, and -ymax.

COMMANDS

zcurve eval curve x
Computes and returns the value of Z(x) for the Z-curve defined by curve, which is a list {lo a b hi}.
zcurve validate curve
Validates that curve defines a valid Z-curve, a list {lo a b hi}, where all four elements are numbers and a <= b.
zcurve object ?option value...?

Creates a Z-curve type with minimum and maximum limits for the for parameters {lo a b hi}. The limits are set using the following options, which can only be set at creation time:

-xmin xmin
-xmax xmax
Sets the minimum and maximum values for a and b.
-ymin ymin
-ymax ymax
Sets the minimum and maximum values for lo and hi.

INSTANCE COMMAND

An instance of zcurve has the following subcommands:

$object eval curve x

Computes and returns the value of Z(x) for the Z-curve defined by curve, which is a list {lo a b hi}.

Note that there is no difference between this call and zcurve eval. In particular, no check is done to verify that curve is valid with respect to this instance's limits.

$object validate curve
This call performs the same checks as zcurve validate, and then verifies that the four parameters {lo a b hi} are within the minimum and maximum limits. On success, returns curve.

ENVIRONMENT

zcurve(n) requires Tcl 8.5 or later.

To use zcurve(n) in a Tcl script, the environment variable TCLLIBPATH must include the parent of the marsutil(n) library directory.
