SYNOPSIS

  • mars icons [namespace]

DESCRIPTION

mars icons [namespace]

mars_icons(1) is a browser for the icon images that are included with marsgui(n). By default, it shows the standard set of marsicons(n) icons; it can also be passed a namespace name to browse the icons from other namespaces.

Icon Namespaces

The marsgui(n) icon set is displayed by default; it consists of a number of icons created using the mkicon(n) command, which allows multiple icons to be created from a single pattern, editable in a text editor, and which also allows icons to contain transparent pixels. These icons have names like ::marsgui::icon::name.
